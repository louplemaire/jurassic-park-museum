# Three.js Catacombs Museum

Immerse yourself in the Catabombs of Paris forbidden to the public.

## Features

- Click on the objects to see an information sheet
- Scroll navigation

## Resource

### Textures

- texturehaven.com
- poliigon.com

### Models

- sketchfab.com

## Instructions

Install dependencies

```npm i```

Launch dev server

```npm run dev```

Build for production

```npm run build```
